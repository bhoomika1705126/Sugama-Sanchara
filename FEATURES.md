## 🎨 Visual Design Principles

### Color Psychology
- **Purple (#667eea)**: Trust + Technology + Innovation
- **Green (#10b981)**: Safety + Success
- **Amber (#f59e0b)**: Warning + Attention
- **Red (#ef4444)**: Critical + Immediate Action

### Typography
- **Headers**: Large, bold, consistent
- **Body**: Clear, readable sans-serif
- **Code**: Monospace for technical display

### Spacing

- 1.5rem padding inside cards
- 0.5rem margins between list items
- Creates breathing room, not cramped
